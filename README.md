## üí° Quick Reference Tables

### Time Complexity Cheat Sheet
```
O(1)       - Hash table operations, Array access
O(log n)   - Binary search, Balanced BST operations
O(n)       - Linear search, Array traversal
O(n log n) - Merge sort, Quick sort, Heap sort
O(n¬≤)      - Bubble sort, Selection sort, Nested loops
O(2^n)     - Recursive Fibonacci, Subset generation
```

### Space Complexity Guide
```
O(1)   - Iterative algorithms, In-place operations
O(log n) - Binary search recursion
O(n)   - Single array/list/hash table
O(n¬≤)  - 2D matrix
```

### Data Structure Selection
```
Fast access by index ‚Üí Array
Fast insertion/deletion ‚Üí Linked List
LIFO operations ‚Üí Stack
FIFO operations ‚Üí Queue
Hierarchical data ‚Üí Tree
Fast search/insert/delete ‚Üí Hash Table
Sorted data with fast search ‚Üí BST
Priority-based processing ‚Üí Heap
```
