## üí° COMMON EXAM TRAPS (AVOID THESE!)

### Language-Specific Pitfalls:
- ‚ùå **C:** Forgetting `\0` null terminator in strings
- ‚ùå **C++:** `substr(start, length)` vs Java `substring(start, end)`
- ‚ùå **Java:** Array index out of bounds, NullPointerException
- ‚ùå **Python:** Indentation errors, mixing tabs/spaces

### Algorithm Traps:
- ‚ùå Off-by-one errors in loops (`i < n` vs `i <= n`)
- ‚ùå Forgetting base cases in recursion
- ‚ùå Choosing wrong algorithm (greedy vs DP)
- ‚ùå Not handling edge cases (empty input, single element)

### Complexity Traps:
- ‚ùå Confusing O(log n) with O(n log n)
- ‚ùå Forgetting space complexity in recursion
- ‚ùå Worst vs average case confusion
